# Contributing

A single-maintainer project under [Refraction](https://refraction.studio) with an opinionated roadmap. Issues are read; response times are not guaranteed.

Discuss any non-trivial change in an issue before writing code. Feature PRs without prior issue discussion are unlikely to land.

Not every feature fits the project. Some PRs will be declined for direction reasons that aren't easy to state in one line.

Bug fixes are accepted on technical merit alone. Design and UI changes may be rewritten to match existing patterns.
